## Features

### Site Features

- Accounts
  * Log in/out and sign up
  * Data changing
  * Permissions: member, staff
- Django Admin (CRUD functionality)
- Trainers Info
- Cards
- Blog (club news)
- Member subscriptions
- Booking system
  
### Desktop Features

- Windows platform-oriented
- Accounts
  * Log in/out
  * Data changing
  * Permissions: member, staff
- Calories control
- Training system
- Booking system
- Staff control panel:
  * Reservations management
  * Subscriptions management
  * Accounts sign up
